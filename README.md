# DeepLearning-Notes

## About
This repository contains notes and examples for the Deep Learning course (DSAI 4302) taught by Thaer Sahmoud.  
It is based on the book *Deep Learning* by Ian Goodfellow, Yoshua Bengio, and Aaron Courville.  

The goal is to provide **easy-to-understand explanations**, code examples, and visualizations for each chapter.  
It is useful for students who want to study Deep Learning in a structured way.

## Chapters Covered
- Chapter 2: Linear Algebra
- Chapter 3: Probability and Information Theory
- Chapter 6: Deep Feedforward Networks
- Chapter 7: Regularization for Deep Learning
- Chapter 8: Optimization for Training Deep Models
- Chapter 9: Convolutional Networks
- Chapter 10: Sequence Modeling: Recurrent and Recursive Networks
- Transformer Model

## Structure
Each chapter folder contains:
- `notes.md` → Short explanations of the main concepts
- `examples.ipynb` → Python code examples for practice
- `images/` → Diagrams or illustrations
